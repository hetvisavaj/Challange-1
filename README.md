# 🌿 EcoSphere 

> **Track, Reduce, and Gamify Your Carbon Footprint.** Make a real impact, one eco-habit at a time.

EcoSphere is an interactive carbon footprint tracker, dashboard, and simulation web application. Users can compute their weekly emissions, compete on a community leaderboard, complete eco-friendly quests, and grow a beautiful, interactive **Virtual Forest** using points earned from green habits.

---

## ✨ Features

### 📊 1. Footprint Calculator
- Estimate your lifestyle footprint across four key dimensions: **Transportation, Home Energy, Diet & Food, and Waste**.
- Visualized progress bar guides you step-by-step.
- Dynamic gauge updates your annual carbon footprint projection (in tons of CO₂e/yr) in real-time as you slide options.

### 🌳 2. My Virtual Forest
- A live, dynamic digital ecosystem representing your environmental contributions.
- Earn **EcoPoints** to plant Conifers and Oaks, unlock Wildflowers and Shrubs, and invite Wildlife (birds, rabbits, deer) into your sanctuary.
- Perform daily watering (+5 XP) with visual raindrop animations and sways.

### 🏆 3. Gamified EcoQuests & Achievements
- Choose from **Daily Habits, Weekly Quests, and Special Events** (e.g., Meatless Mondays, Commute Cycling, Zero Plastic).
- Earn XP to level up your rating (from *Eco Explorer* to *Planet Hero*).
- Unlock special achievement badges like **Carbon Analyst**, **Streak Builder**, and **Carbon Slasher**.

### 🎛️ 4. Impact Simulator
- Adjust sliders to model hypothetical improvements in your lifestyle (e.g., reducing drive distance by 50%, shifting to clean energy).
- Instantly view carbon reductions, utility bill savings, and mature tree offset equivalents.

### 👥 5. Community Leaderboard & Eco-Shop
- Compare ranks and badges against other global Eco Warriors.
- Redeem points in the Eco-Shop to secure premium species and carbon offset certificate achievements.

### 📱 6. Fully Responsive Mobile UI
- Features a premium slide-out sidebar menu drawer on mobile.
- Visual components, tables, progress bars, and forms stack and scale beautifully on any viewport width (down to 320px).

## 🛠️ Tech Stack

- **Core**: Vanilla HTML5, CSS3, ES6+ Javascript
- **Visuals**: Scalable Vector Graphics (SVG) with CSS keyframes
- **Icons**: Font Awesome (CDN)
- **Fonts**: Google Fonts (*Plus Jakarta Sans*)

## 🚀 Local Setup

Since EcoSphere is built on static web technologies, it runs directly in any modern browser without compiling.

### Method 1: Using Python (Recommended)
If you have Python installed, run this in the root project folder to spin up a local server:
```bash
python -m http.server 8000
```
Then open `http://localhost:8000` in your web browser.

### Method 2: Standard Browser Run
Simply double-click the `index.html` file to open it in your browser.
